# Pet Persona Lab 🐾

一个仿 MBTI 交互流程设计的「宠物性格测试网站」，面向 20-30 岁女生用户，主打轻松有趣 + 结果报告专业化。

## 功能亮点

- 16 道情境题，覆盖外向/内向、理性/感性、计划/灵活等性格维度。
- 动态进度条 + 选项卡片交互，支持回看结果。
- 结果页提供：
  - 宠物人格类型（四维编码）
  - 画像关键词与说明
  - 维度得分条
  - 关系、生活、压力管理建议
- 支持一键重新测试。

## 本地运行

```bash
python3 -m http.server 4173
```

打开 `http://localhost:4173`。
